# 🔍 AniScanner – یه ابزار باحال برای اسکن پورت ۴۴۳

سلام! این یه ابزار ساده ولی خفنه که میاد پورت 443 رو اسکن می‌کنه (همون HTTPS) و SNI رو هم چک می‌کنه. یه سرور پایتون توی بک‌اند کار می‌کنه، پس اگه مرورگر رو ببندی، اسکن ادامه داره، نگران نباش.

## چیزایی که بلده 😎

- اسکن آی‌پی تکی، دامنه، رنج (`192.168.1.1-192.168.1.10`) و CIDR (`23.44.229.0/24`)
- یه فرمت خاص: `IP:SNI` مثلاً `8.8.8.8:google.com`
- آپلود فایل JSON یا TXT (خودش تشخیص می‌ده)
- دانلود نتایج به دو حالت:
  - ساده: فقط آی‌پی‌ها (هر خط یکی)
  - پیشرفته: `IP (SNI) - Ping: Xms, Status: OPEN`
- لاگ زنده تو مرورگر (موفق و ناموفق)
- نشون دادن پیشرفت اسکن تو ترمینال (هر ۱۰٪)
- قابلیت توقف اسکن وسط کار

## نصب و راه‌اندازی (تو ترموکس یا لینوکس)

**اول پیش‌نیازها:**
```bash
pkg update && pkg upgrade
pkg install python
```

بعد دانلود پروژه:

```bash
git clone https://github.com/ForExampleZERO/ApiScanner.git
cd ApiScanner
```

نصب کتابخونه‌ها:

```bash
pip install flask websockets
```

اجرای سرور:

```bash
python main.py
```

حالا برو تو مرورگر آدرس زیر رو وارد کن:

```
http://localhost:5000
```

اگه می‌خوای از گوشی دیگه یا کامپیوتر دیگه وصل شی، به جای localhost از آی‌پی دستگاهت استفاده کن.

چطور استفاده کنم؟

1. توی کادر متنی یا یکی از اینا رو بنویس (هر خط یکی):
   · google.com
   · 1.1.1.1
   · 8.8.8.8:cloudflare.com
   · 192.168.1.1-192.168.1.10
   · 23.44.229.0/24
2. یا یه فایل JSON مثل این آپلود کن:
   ```json
   [
     "23.44.229.0/24",
     "1.1.1.1",
     "google.com",
     "8.8.8.8:cloudflare.com"
   ]
   ```
3. تعداد همزمانی و تایم‌اوت رو مطابق با قدرت گوشیت تنظیم کن (اگه گوشیت ضعیفه تعداد همزمانی رو کم کن).
4. بزن شروع اسکن و ببین چه خبره.
5. بعد از تموم شدن، دکمه دانلود نتایج رو بزن و یکی از دو حالت رو انتخاب کن.
